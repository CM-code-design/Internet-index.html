Overview

The internet is a worldwide system of interconnected computer networks and devices that can communicate using established sets of protocols.

History

For the record, no one RUNS the internet, it's an anarchy. It does have creators, "Vint" Cerf,Bob Khan and more. Originally funded by the D.O.D, in 1983, The internet was created with the goal of being able to have a network that would allow users on a computer talk to other computers farther away. This achievement made them realize that the messages sent back and forth could be routed to different connections. This was beneficial because if someone like a hacker were to destroy a part of the function, there would still be a way for the message to get from point A to point B. Although they achieved their goal, there was still much work to be done. When HTTPS were created communications began to improve because different computer platforms could connect to the same internet sites.

In modern times

Though there is much more history to the creation of the internet, like IP addresses and DNS, the internet of today still manages to grow, connect and give information to those who seek it.
