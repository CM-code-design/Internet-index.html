What is a router?

A router is a device that connects other devices to the internet which allows them to communicate with each other.

How routers work

When the router receives connection from either an ethernet cable or a wireless signal (wifi), the router distributes the connection throughout multiple devices using the given connection options.

Why do routers need redundancy?

For redundancy to occur, you must have a router. Redundancy acts as a backup source for connection to help guarantee limitless internet access and functionality. For example, if a point of access in a network goes out, the network will immediately find a new path to reroute traffic. Redundancy helps to keep the speed of the network up to date.

Packets

Packets are bundles of information sent over a network from one device to another. We often use packets in everyday life. A common example would be messages! Messages are sent as packages because they are an efficient transmission of data. If you look at a messages router logs, you can sometimes see where they bounced from one router to another before they get to their destination.
